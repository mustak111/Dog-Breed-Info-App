## 2. Overview / Introduction

**Dog-Breed-Info-App** is a frontend-focused, API-driven console-based application that allows users to explore different dog breeds and learn detailed information about each breed.

The application fetches real-time data from a public Dog API and presents it in a clean and user-friendly interface. It is designed to help users quickly understand dog characteristics such as origin, temperament, lifespan, size, and appearance.

This project was built to:
- Practice real-world API integration
